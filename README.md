git clone <>

🛠 Tech Stack Used

Next.js (App Router)

TypeScript

Tailwind CSS

DaisyUI

Recharts

Lucide React

🏗 Architecture Decisions

Used Next.js App Router for better routing and layout management

Component-based structure for reusability and clean separation of concerns

Tailwind CSS with DaisyUI for fast, consistent UI development

Static mock data to focus on UI and interaction


✅ Assumptions Made

Backend and authentication are out of scope

Data is static and non-persistent

Dashboard is for admin/analytics view only

Focus is on UI/UX rather than business logic
